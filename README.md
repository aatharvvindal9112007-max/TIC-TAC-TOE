# 🎮 Tic-Tac-Toe CLI (Python)

A fully functional command-line Tic-Tac-Toe game built using Python.  
This project demonstrates core programming concepts such as game logic, input validation, state management, and control flow.

---

## 📌 Overview

This is a two-player Tic-Tac-Toe game where players take turns marking spaces on a 3×3 grid.  
The game ensures valid inputs, prevents illegal moves, and correctly determines the winner or a draw.

---

## ✨ Features

- 👥 Two-player gameplay (Player vs Player)
- 🎲 Randomized starting player
- ✅ Robust input validation (handles invalid and non-numeric inputs)
- 🚫 Prevents overwriting occupied cells
- 🧠 Accurate win detection using predefined winning combinations
- 🔄 Replay functionality with board reset
- 🖥️ Clean and readable CLI interface

---

## 🧠 Concepts Demonstrated

This project showcases:

- Functions and modular code design
- Loops and control flow
- Conditional logic
- List manipulation

- Game state management
